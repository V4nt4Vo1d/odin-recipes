About This Project

This website is a small, focused project built as part of The Odin Project — Foundations. The goal is to practice clean, semantic HTML, light CSS, accessible content, and a tidy project structure while using Git/GitHub for version control.

Scope & Objectives

  • Build a minimal multi‑page site: a homepage plus individual recipe pages.
  • Use semantic elements (<header>, <main>, <article>, <nav>, <footer>) for structure.
  • Demonstrate lists for ingredients and steps, and include appropriately described images.
  • Keep styles simple and readable without any frameworks.

Tech Stack

  • HTML5 for structure and semantics
  • CSS3 for basic styling and layout
  • Git & GitHub for source control

Project Structure

odin-recipes/
├─ index.html         # Homepage
├─ about.html         # This page
├─ recipes/           # Individual recipe pages
│  ├─ cookie.html

└─ images/            # Recipe images

Accessibility Notes

  • Informative alt text for images.
  • Headings arranged in a logical outline (H1 → H2 → H3...).
  • Clear link text and visible focus styles.
